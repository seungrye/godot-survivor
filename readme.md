Key Feature
-----------

* 시간이 지날수록 적의 수가 기하급수적으로 증가
* 공격은 자동, 플레이어는 이동/회피/경로 선택
* 선택 기반 성장(Build-up)이 플레이의 운명 결정
  * 레벨업 시 스킬 트리 분기:
	* 공격형 / 보조형 / 범위 증가 중 선택하여 성장
* 시간 생존 보상
* 특정 시간대에 등장하는 ‘위기 구간’ 극복
* 특정 조건 누적 시 “각성(Ascension)” 발생
  * 회피 기반 플레이 누적 시 “쉐도우 대시 진화”
  * 근접 딜 위주 플레이 시 “레이지 모드” 활성화
* 환경 오브젝트를 활용한 전투 변주
  * 폭발하는 버섯
  * 밀어낼 수 있는 바위
  * 물속 지대: 속도 감소 or 감전 시 데미지 강화
* 일정 시간마다 발생하는 “필드 이벤트”
  * 폭풍우: 랜덤 지점 번개
  * 밤/낮 변화: 등장 몬스터 타입 변화
* 적이 단순히 몰려오는 것이 아니라, 웨이브 패턴이 존재
  * 포위형 웨이브
  * 돌격형 웨이브
  * 스나이핑 원거리 웨이브
* 플레이어를 보조하는 소환물
  * 자율 공격 드론
  * 힐링 토템
  * 자동 루팅 페어리
* 일정 시간/킬수 충전 후 궁극기 발동
  * 카메라 줌아웃 + 화면 전체 특수 이펙트
  * 예:
	* 초신성 폭발
	* 타임스톱 후 광역 참격
	* 맵 전체에 지속 마법 지대 생성
* 영구 능력 트리
* 무기 해금 · 신규 캐릭터 해금
* 도전 과제 및 업적 보상

Core Loop
---------

* 적 회피 & 자동 공격
* 적 처치 → 경험치 획득
* 레벨업 선택(무기/스킬)
* 강화된 스킬로 더 많은 적 처치
* 시간 경과 → 난이도 증가
* 보스 등장 → 생존 여부 판정
* 게임 종료 → Meta Progression 보상

Dev Roadmap
-----------

Core Prototype
==============

* 기본 이동/충돌/자동 공격
* 1종 무기 & 1종 몬스터
* 간단한 웨이브 생성
* 레벨업 UI 폴리싱

콘텐츠 확장
=========

* 무기 6종
* 패시브 스킬 6종
* 몬스터 10종
* 맵 2종
* 진화 시스템 추가


차별화 기능 구현
==============

* Dynamic Field Event
* 궁극기 시스템
* 환경 상호작용
* 동료 시스템
* 영구 능력 업그레이드

Top-level UX Flow
-----------------

```
[게임 실행]
   ↓
[타이틀 화면]
   ├─ Start Game
   │     └─ 캐릭터 선택
   │             └─ 맵 선택
   │                     └─ 게임 플레이
   ├─ Meta Upgrade (영구 업그레이드)
   ├─ Codex (무기/스킬/몬스터 도감)
   ├─ 옵션(그래픽/사운드/키 설정)
   └─ 종료
```

Gameplay UX Flow
----------------

```
[HUD 표시]
   ├─ 플레이어 HP/스킬 쿨다운
   ├─ 경험치 게이지
   ├─ 현재 시간(Timer)
   ├─ 미니맵(선택 옵션)
   └─ 무기 슬롯(현재 보유 무기 레벨 표시)

[레벨업 발생]
   ↓
[Skill/Weapon 선택 UI]
   ↓
[전투 지속]
   ↓
[특정 시간 도달]
   ↓
[보스 등장 UI + 경고 표시]
   ↓
[전투 종료]
   ├─ Victory 화면
   └─ Game Over 화면

[결과 정산 UI]
   ├─ 생존 시간
   ├─ 누적 적 처치
   ├─ 최종 빌드 요약
   ├─ Meta Currency 보상
   ↓
[메인 메뉴로 복귀]
```
